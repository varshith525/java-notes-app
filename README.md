# Notes App (Java File I/O)

This is a simple Java project that lets you:
- Save text notes into a file
- Read notes from the file

## How to Run
1. Compile:
   javac NotesApp.java
2. Run:
   java NotesApp

## Features
- Uses FileWriter to save notes
- Uses BufferedReader to read notes
- Handles exceptions
